# GSD SaaS Creator

The `gsd-saas-creator` is a standalone Agent Skill designed to automate the end-to-end process of creating and deploying a SaaS application. It bridges the gap between design (Google Stitch) and production (Hostinger/Firebase).

## Features
- **Design DNA Extraction**: Automated retrieval of visual tokens from Google Stitch.
- **Autonomous Orchestration**: Multi-agent workflow involving PM, Engineering, QA, and DevOps roles.
- **Safe Deployment**: Built-in Halt Gates for user approval at critical stages.
- **Modern Tech Stack**: Integrated with React, Firebase, and GitHub Actions.

## Setup
1. Clone this repository into your agent's skills directory.
2. Ensure you have the necessary API keys configured.
3. Use the `SKILL.md` as the entry point for orchestration.
